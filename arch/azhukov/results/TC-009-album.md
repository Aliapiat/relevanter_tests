## Общие проверки каждого теста

Каждый тест проверяет:
1. API-параметр корректен в URL запроса (source=estaff + фильтр)
2. Нет NaN/undefined в параметрах
3. Ответ содержит items[] с кандидатами (total > 0)
4. Данные каждого кандидата соответствуют фильтру
5. Таблица рендерится в UI (getByRole('table') visible)
6. Строки таблицы > 0
7. Скриншоты: t0 (сразу), t1-t4 (по секунде), t5-final (через 5 секунд)
