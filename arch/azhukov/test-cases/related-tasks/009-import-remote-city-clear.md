# Тест-кейсы: Импорт вакансии — убрать город для удалённого формата

## ПОЗИТИВНЫЕ СЦЕНАРИИ

### TC-001: Импорт вакансии с Удалёнкой → город пустой
**Предусловия**: Вакансия на HH с форматом "Удалёнка"
**Шаги**: Импортировать вакансию
**Ожидаемый результат**: Поле город в "Формат работы" пустое

### TC-002: Импорт вакансии с Офис → город заполнен
**Предусловия**: Вакансия на HH с форматом "Офис" и городом "Москва"
**Шаги**: Импортировать вакансию
**Ожидаемый результат**: Поле город = "Москва"

### TC-003: Смена Удалёнка → Офис → выбор города → метро/адрес появляются
**Предусловия**: Импортированная вакансия с Удалёнкой, город пустой
**Шаги**:
1. Сменить формат на "Офис"
2. Выбрать город "Москва"
**Ожидаемый результат**: Поля метро и адрес появляются

## НЕГАТИВНЫЕ СЦЕНАРИИ

### TC-101: Импорт с Гибрид → город заполняется
**Предусловия**: Вакансия на HH с форматом "Гибрид"
**Шаги**: Импортировать
**Ожидаемый результат**: Город заполнен (как раньше)
