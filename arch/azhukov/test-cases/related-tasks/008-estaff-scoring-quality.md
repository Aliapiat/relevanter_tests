# Тест-кейсы: E-Staff поиск — качество AI-оценки (TASKNEIROKLYUCH-624)

## ПОЗИТИВНЫЕ СЦЕНАРИИ

### TC-001: Пустые резюме не попадают в scoring
**Предусловия**: E-Staff возвращает кандидата без body и experience
**Шаги**: Запустить E-Staff поиск
**Ожидаемый результат**: Кандидат с пустым резюме не получает AI-оценку (или получает 0 с пометкой "Недостаточно данных")

### TC-002: QA и Senior QA — exact match по должности
**Предусловия**: Вакансия "QA", кандидат "Senior QA"
**Шаги**: Запустить поиск
**Ожидаемый результат**: positionMatch = "exact" или "partial_same_domain" с высоким баллом

### TC-003: AI не оценивает критерии вне вакансии
**Предусловия**: Вакансия QA без требования английского
**Шаги**: Запустить поиск, проверить AI-комментарии
**Ожидаемый результат**: AI не упоминает английский в негативных фактах

## НЕГАТИВНЫЕ СЦЕНАРИИ

### TC-101: Кандидат с навыками получает адекватную оценку
**Предусловия**: Кандидат с навыками тестирования, автоматизации, Playwright
**Шаги**: Запустить поиск по вакансии QA
**Ожидаемый результат**: Оценка ≥70%, AI-комментарий отмечает наличие навыков
