# Тест-кейсы: Поиск/Оценка — сломалась оценка по релевантности (TASKNEIROKLYUCH-623)

## ПОЗИТИВНЫЕ СЦЕНАРИИ

### TC-001: userId пробрасывается в generateComment
**Предусловия**: Авторизованный пользователь с привязанным HH-аккаунтом
**Шаги**:
1. Запустить поиск кандидатов по вакансии через search-with-relevance
2. Проверить логи relevanter-back
**Ожидаемый результат**:
- В логах `generateComment` видно `userId: <number>` (не null)
- `getResumeByUrl` вызывается с userId

### TC-002: Полное резюме загружается для scoring
**Предусловия**: Поиск с AI-оценкой
**Шаги**:
1. Запустить поиск
2. Проверить логи — размер контекста для AI
**Ожидаемый результат**:
- `Resume context for AI:` содержит `contextLength > 1000` (полное резюме, не шапка)
- В контексте есть `experience[].description` (описания опыта)

### TC-003: Оценка учитывает описание опыта (foundInBody)
**Предусловия**: Кандидат с релевантным описанием опыта
**Шаги**:
1. Запустить поиск по вакансии Data Science/ML
2. Посмотреть AI-комментарий кандидата с 3+ годами опыта ML
**Ожидаемый результат**:
- Оценка ≥70% (Высокое соответствие)
- В AI-комментарии упоминаются конкретные проекты/технологии из описания опыта
- `foundInBody` содержит навыки (не только `foundTagsOnly`)

### TC-004: Одинаковая оценка на dev и QA
**Предусловия**: Одинаковые вакансии на обоих стендах
**Шаги**:
1. Поиск на dev: вакансия 179 (Data Science/ML)
2. Поиск на QA: вакансия 2 (Data Science/ML)
3. Сравнить оценки одних и тех же кандидатов
**Ожидаемый результат**: Оценки совпадают (±10%)

## НЕГАТИВНЫЕ СЦЕНАРИИ

### TC-101: getResume фейлится — fallback на краткое резюме
**Предусловия**: HH API недоступен для полных резюме
**Шаги**:
1. Симулировать ошибку getResume (timeout/403)
2. Запустить поиск
**Ожидаемый результат**:
- Warning в логах: "Failed to fetch full resume"
- Оценка всё равно генерируется (по краткому резюме)
- Не крашит весь поиск

### TC-102: Кандидат с пустым резюме — низкая оценка
**Предусловия**: Резюме без описания опыта (только шапка)
**Шаги**:
1. Найти такого кандидата
2. Проверить оценку
**Ожидаемый результат**:
- Оценка <60%
- AI-комментарий отмечает недостаток информации
