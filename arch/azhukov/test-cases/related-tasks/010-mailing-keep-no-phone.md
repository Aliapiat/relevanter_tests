# Тест-кейсы: Рассылки — не удалять кандидатов без телефона (TASKNEIROKLYUCH-530)

## ПОЗИТИВНЫЕ СЦЕНАРИИ

### TC-001: Кандидат без телефона добавляется в рассылку
**Предусловия**: Кандидат без открытых контактов (нет телефона)
**Шаги**: Выбрать кандидата → Добавить в рассылку
**Ожидаемый результат**: Кандидат добавлен, без ошибок

### TC-002: Кандидат с телефоном добавляется в рассылку (регресс)
**Предусловия**: Кандидат с открытым телефоном
**Шаги**: Выбрать → Добавить
**Ожидаемый результат**: Кандидат добавлен (как раньше)

### TC-003: Массовое добавление — все кандидаты попадают
**Предусловия**: 10 кандидатов, часть без телефона
**Шаги**: Выбрать всех → Добавить в рассылку
**Ожидаемый результат**: Все 10 добавлены

## НЕГАТИВНЫЕ СЦЕНАРИИ

### TC-101: Рассылка через мессенджер HH работает без телефона
**Предусловия**: Кандидат без телефона в рассылке
**Шаги**: Запустить рассылку
**Ожидаемый результат**: Сообщение отправляется через мессенджер HH
