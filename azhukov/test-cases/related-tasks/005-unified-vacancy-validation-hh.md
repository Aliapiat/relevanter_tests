# Тест-кейсы: Единая валидация вакансии + поиск

**База:** таблица в `info/sprint4/Обновлённые тест кейсы - Валидация вакансии (1).csv` и `001_task-hh-export-validation/test-cases.md`.

## Позитивные

### TC-001: Название ровно 100 символов — сохранение и экспорт

**Предусловия:** вакансия с привязкой к HH.  
**Шаги:** ввести название длиной 100 символов (допустимые символы), сохранить, экспортировать.  
**Ожидание:** сохранение без предупреждения о «внутреннем 200»; экспорт без ошибки длины `name`.

### TC-002: Сумма трёх блоков описания 150 символов — минимальный допустимый экспорт

**Шаги:** распределить 150 символов между тремя полями, сохранить, экспорт.  
**Ожидание:** нет ошибки минимума; сообщения используют **150**, не **200**.

### TC-003: Сумма описания около лимита 10 000 — сохранение и экспорт

**Шаги:** заполнить три поля так, чтобы после склейки было 10 000 (и отдельно кейс 9993 — регрессия «обрезки»).  
**Ожидание:** поведение документировано; нет молчаливой обрезки без уведомления.

## Негативные / регрессия поиска

### TC-101: Длинное описание вакансии — поиск кандидатов

**Предусловия:** вакансии с длинным описанием (референс: id **184**, **85**; ~2449/2498 символов — поиск падал).  
**Шаги:** открыть поиск с фильтром по этой вакансии / по тексту описания.  
**Ожидание:** нет **414** / **431**; при техническом ограничении — понятное сообщение, не обрыв запроса.

### TC-102: География — регион целиком с исключением города

**Шаги:** выбрать область (напр. Московская), снять один населённый пункт, сохранить фильтр, выполнить поиск.  
**Ожидание:** нет **431 Request Header Fields Too Large**; фильтр передаётся компактно.

### TC-103: Ошибка HH при отсутствии кредитов

**Шаги:** валидная вакансия, аккаунт HH без кредита на публикацию, экспорт.  
**Ожидание:** сообщение на русском о невозможности размещения / кредитах, не сырой `quota_exceeded`.

## Символы и разметка

### TC-201: Управляющие символы (кроме таб и перевода строки)

**Шаги:** вставить запрещённые control chars в название/описание.  
**Ожидание:** валидация до сохранения; единое сообщение.

### TC-202: Недопустимые HTML-теги / ссылки

**Шаги:** вставить `<script>`, стили, длинный URL в описание.  
**Ожидание:** согласно политике продукта — блок или предупреждение до экспорта.
