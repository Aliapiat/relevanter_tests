# Тест-кейсы: Валидация полей вакансии для экспорта на HH.ru

> Валидация происходит в `relevanter-back/services/hhVacancyMappingService.js` перед отправкой на HH API.
> Фронтенд показывает предупреждения (soft validation), но не блокирует сохранение вакансии.

---

## TC-001: Название вакансии (name)

### TC-001-01: Название пустое
**Предусловия**: Вакансия без заполненного названия
**Шаги**: Попытка экспорта на HH.ru
**Ожидаемый результат**: Ошибка валидации "Название вакансии обязательно", экспорт заблокирован

### TC-001-02: Название превышает 100 символов
**Предусловия**: title = 101 символ (допустимо в нашей системе, max=200)
**Шаги**: Попытка экспорта на HH.ru
**Ожидаемый результат**: Ошибка "Название не должно превышать 100 символов для HH.ru (сейчас: 101)"

### TC-001-03: Название ровно 100 символов
**Шаги**: title = 100 символов, экспорт
**Ожидаемый результат**: Валидация пройдена

### TC-001-04: Название 1 символ
**Шаги**: title = 1 символ, экспорт
**Ожидаемый результат**: Валидация пройдена

### TC-001-05: Название 150 символов, сохранение без экспорта
**Шаги**: Сохранить вакансию с title=150 символов (без публикации на HH)
**Ожидаемый результат**: Сохранение проходит (HH-валидация не применяется)

## TC-002: Итоговое описание (description = companyDescription + description + socialPackageText)

### TC-002-01: Все 3 поля пустые
**Шаги**: Оставить companyDescription, description, socialPackageText пустыми, экспорт
**Ожидаемый результат**: Ошибка "Описание вакансии обязательно"

### TC-002-02: Итоговое описание менее 150 символов (stripped HTML)
**Предусловия**: companyDescription = 50 символов, description = 50 символов, socialPackageText = 40 символов (итого 140)
**Шаги**: Экспорт на HH.ru
**Ожидаемый результат**: Ошибка "Описание должно содержать не менее 150 символов (сейчас: 140)"

### TC-002-03: Итоговое описание ровно 150 символов
**Предусловия**: Сумма текста 3 полей (без HTML) = 150 символов
**Шаги**: Экспорт на HH.ru
**Ожидаемый результат**: Валидация пройдена

### TC-002-04: Итоговое описание превышает 10000 символов
**Предусловия**: description = 8000 + companyDescription = 2000 + socialPackageText = 1000 (итого 11000)
**Шаги**: Экспорт на HH.ru
**Ожидаемый результат**: Ошибка "Описание не должно превышать 10000 символов (сейчас: 11000)"

### TC-002-05: Подсчёт длины по тексту без HTML-тегов
**Предусловия**: description = `<b>Текст</b> <ul><li>пункт</li></ul>` (реальный текст ~12 символов)
**Шаги**: Экспорт на HH.ru
**Ожидаемый результат**: Длина считается по stripped тексту (12), а не по полной строке с тегами

### TC-002-06: Запрещённые HTML-теги
**Предусловия**: description содержит `<script>alert(1)</script>`, `<style>`, `<iframe>`, `onclick=`
**Шаги**: Экспорт на HH.ru
**Ожидаемый результат**: Запрещённые теги удалены/экранированы перед отправкой на HH

### TC-002-07: Разрешённые HTML-теги сохраняются
**Предусловия**: description с тегами `<b>`, `<i>`, `<ul>`, `<li>`, `<ol>`, `<p>`, `<br>`
**Шаги**: Экспорт на HH.ru
**Ожидаемый результат**: Разрешённые теги сохранены в итоговом JSON

### TC-002-08: Управляющие символы
**Предусловия**: description содержит символы \x00-\x08, \x0B, \x0C, \x0E-\x1F
**Шаги**: Экспорт на HH.ru
**Ожидаемый результат**: Управляющие символы удалены (кроме \t и \n)
